Lets Learn To Setup a Blog Using Hugo

Mon, Dec 28, 2015 - Read in 2 Min
IMP - This post is based on WINDOWS version of Hugo. Hugo is a static website generator. Okay, our first step is to have Hugo and Git installed on your PC. Once installed, make sure it is kept in your system path. Run cmd on you PC and set the location where you want to work on the blog.After doing that that type the following command :- hugo new site [Folder name you want to create] {for example :- hugo new site learn} A new folder will be created where you set the location.
